The default keymappings in showtime are as follows:

End
Stop playback
F1
Context menu
F2
Play / Pause
F3
Switch view in browser
F5
Volume Down
F6
Volume Up
F7
Toggle mute
F10
Quit
F11
Seek backward
F12
Seek forward
Space
Select (enqueue tracks on playlist without starting)
Menu key
Go to mainmenu
Page up
Previous track
Page down
Next track

The mappings can be changed under Settings / Keymapper.
