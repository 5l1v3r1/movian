Starting Showtime
Just start it by typing 'showtime' on the command prompt.

See the configuration chapter for information how to configure it.

The current settings will be stored in .hts/showtime in the current user's home directory.

Building Showtime from source
If you want to build Showtime from source, visit this page. Please notice that wiki development site only reflects the work in 'trunk' (current development version). The instructions should not deviate much should you want to build a released version from source, but you never know.
